## fruits-seed

### 概要

シーダーのおべんきょう。  
くわしくは[こちらのPR](https://github.com/hrtg-yamazaki/fruits-seed/pull/1)にどうぞ。

### 動作確認

http://localhost:8310

- 1回目(環境構築)

```
$ git clone git@github.com:hrtg-yamazaki/fruits-seed.git

$ cd fruit-seed

$ make init
```

- 2回目以降

```
$ make up
```

- seeder実行

```
$ make seed
```

その他コマンドは[こちら](https://github.com/hrtg-yamazaki/fruits-seed/blob/main/Makefile)
